# NEXUS 

NEXUS is a full-stack AI-powered knowledge sharing platform designed for technology enthusiasts to create, discover and share technical knowledge.

Frontend : https://nexus-frontend-x21k.onrender.com/
Backend : https://nexus-1efg.onrender.com

## Features

- User Registration and Login
- JWT Based Authentication
- Create and Explore Blogs
- AI Powered Content Assistance
- Search Functionality
- User Profiles
- Responsive Modern UI




## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- FastAPI
- Python
- MongoDB
- JWT Authentication

### AI
- Google Gemini API

### Deployment
- Render
